Aperçu

Présentation

Castle Clash intègre un mini-jeu 2048 dans lequel les tuiles représentent des niveaux de héros (affichés sous la forme « Lv.X »). Le but est de fusionner ces tuiles pour atteindre la tuile maximale, en effectuant jusqu'à 1000 mouvements sur une journée.

Le bot automatise entièrement cette tâche : à chaque tour, il prend une capture d'écran du jeu, la lit avec EasyOCR pour reconstruire l'état de la grille 4×4, calcule le meilleur coup grâce à l'algorithme Expectimax, puis simule la frappe clavier correspondante via PyAutoGUI. Un délai aléatoire gaussien entre chaque action lui donne un comportement plus naturel.

Fonctionnalités

Capture d'écran automatique

Déclenche une capture dans Castle Clash via une touche simulée, puis récupère et renomme le fichier généré.

Lecture OCR de la grille

Utilise EasyOCR pour détecter les textes « Lv.X » sur la capture, puis associe chaque valeur à sa case grâce aux coordonnées configurées.

Algorithme Expectimax

Calcule le meilleur coup en explorant l'arbre de jeu avec une profondeur de 2, en modélisant les apparitions aléatoires de nouvelles tuiles (90 % de chances pour un 2, 10 % pour un 4).

Simulation clavier

Envoie la direction choisie (Z/Q/S/D) directement au jeu via PyAutoGUI, sans interaction manuelle.

Délai aléatoire

Attend un temps variable entre chaque coup (distribution gaussienne autour de 90 ms ± 20 ms) pour simuler un comportement humain.

Affichage terminal

Affiche la grille avant et après chaque coup côte à côte dans le terminal, avec l'action effectuée et le temps d'attente.

Stack technique

Le projet est entièrement en Python. La lecture de la grille repose sur EasyOCR pour la reconnaissance de texte sur les captures d'écran. La configuration de la zone de capture et des coordonnées de chaque case est stockée dans un fichier JSON pour faciliter l'adaptation à différentes résolutions. La prise de décision utilise l'algorithme Expectimax avec une table heuristiques précalculées pour chaque ligne possible (65 536 entrées). Les actions sont envoyées au jeu via PyAutoGUI.

Python EasyOCR PyAutoGUI Expectimax JSON

Démarche

Lecture de la grille par OCR

La première difficulté était de lire l'état du jeu depuis une capture d'écran. Les tuiles dans Castle Clash affichent des textes du type « Lv.3 » plutôt que de simples chiffres. J'ai utilisé EasyOCR pour détecter ces textes, puis filtré uniquement ceux contenant « Lv » pour ignorer le reste de l'interface. Chaque valeur détectée est ensuite associée à la bonne case de la grille grâce aux coordonnées configurées manuellement dans config.json.

Configuration des coordonnées

Pour relier les positions détectées par OCR aux cases de la grille, j'ai développé un script config.py qui affiche les coordonnées de chaque élément détecté avec son numéro. Il suffit d'exécuter ce script une fois et de renseigner les coordonnées dans config.json pour calibrer le bot à sa propre résolution d'écran.

Algorithme Expectimax et heuristique

Pour la prise de décision, j'ai réutilisé et adapté l'implémentation Expectimax de cczhong11. L'algorithme explore les 4 directions possibles en profondeur 2, en modélisant les cases aléatoires comme une espérance (90 % tuile 2 / 10 % tuile 4). L'heuristique évalue chaque état via une table précalculée de 65 536 entrées qui prend en compte le nombre de cases vides, les fusions possibles et la monotonie des lignes.

Boucle principale et simulation des touches

Une fois le meilleur coup déterminé, PyAutoGUI envoie la touche correspondante (Z, Q, S ou D) directement au jeu. La boucle principale effectue jusqu'à 1000 mouvements, avec un délai aléatoire gaussien entre chaque action pour éviter une détection comme bot. La grille avant et après chaque coup est affichée dans le terminal pour suivre la progression.

Configuration

01

Dans config.json, renseigner les coordonnées de la zone de capture sous la forme [x, y, largeur, hauteur], le coin haut-gauche de l'écran étant [0, 0].

02

Exécuter config.py et choisir l'option 1 pour prendre une capture de référence et vérifier le cadrage.

03

Exécuter config.py et choisir l'option 2 : les coordonnées de chaque case détectée s'affichent. Les renseigner dans config.json selon leur position dans la grille.

Utilisation

01

Ouvrir Castle Clash et naviguer jusqu'au mini-jeu 2048.

02

Lancer main.py. Le bot démarre après 5 secondes et joue automatiquement jusqu'à 1000 coups.
